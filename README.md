# JS Fundamentals Training

Este repositorio nace con la intención de practicar, recuperar la confianza y volver a tener ganas de programar por cuenta propia.

Lo considero como un hijo, porque lo he empezado desde cero, con la IA como tutor. Está pensado para personas que ya tienen bases mínimas en JavaScript y quieren hacer una transición hacia el aprendizaje de backend.

También puede servir para quienes sienten que en la universidad solo se ven los temas por encima y no se profundiza tanto. Además, muchas veces te obligan a usar la IA solo para cumplir y obtener una calificación, porque piden cosas en muy poco tiempo.

## Métodos de arrays

Aunque leer la documentación puede ser tardado, se recomienda revisarla al menos para darse una idea general, ya que durante el proyecto se utilizan métodos como:

- `.map()`
- `.filter()`
- `.reduce()`
- `.push()`

Documentación recomendada:  
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array

## Backend con Express.js

Una de las ventajas de JavaScript es la creación de APIs o REST APIs. Estas se encargan de manejar todo lo que no vemos que sucede detrás de lo que observamos como usuarios de una página o aplicación web.

Para iniciar con Express.js, se recomienda revisar la documentación oficial:

https://expressjs.com/en/5x/api.html

En esta parte también se aprende a levantar un servidor de manera local. Para probar las rutas, se sugiere utilizar aplicaciones como **Postman** o extensiones de VS Code como **Thunder Client**.

> En esta etapa todavía no se utiliza nodemon ni bases de datos.
