# JS Fundamentals Training

Ejercicios básicos de JavaScript enfocados en:

- Arrays (map, filter, reduce)
- Funciones
- Lógica de programación

## Funciones implementadas

- getEvenNumbers
- getOddNumbers
- doubleNumbers
- countLetters

Todo esto es para practicar o recuperar la confianza y las ganas al querer programar por cuenta propia.
